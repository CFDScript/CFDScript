CFDScript 

A JavaScript library for solving Partial Differential Equations (PDE) using the Finite Element Method (FEM)

CFDScript is a JavaScript library for solving Partial Differential Equations (PDE) using the Finite Element Method (FEM). It allows you to create and run browser-based simulations of physics and engineering problems. CFDScript also provides visualization tools to plot the results of your simulations.

Download

CFDScript is still under development and new features and improvements are constantly being added. You can download the latest version of CFDScript from the Git repository: https://github.com/nikoscham/CFDScript.

Usage Example

Here some example code demonstrating how to use the library to solve a 2D Laplace equation. The following scripts can be embedded to your HTML page. Initially we define the problem parameters: 

// Script1: Problem parameters definition
let u = []; // Initialize solution array
const nex = 5; // Number of elements in x-direction
const ney = 5; // Number of elements in y-direction
const xlast = 1; // Max x-coordinate
const ylast = 1; // Max y-coordinate
const boundaryConditions = {
  neumannTop: true, // Set a Neumann boundary condition at the top of the domain
  neumannBottom: false, // Set a Neumann boundary condition at the bottom of the domain
  neumannLeft: false, // Set a Neumann boundary condition at the left side of the domain
  neumannRight: false, // Set a Neumann boundary condition at the right side of the domain
  dirichletTop: false, // Set a Dirichlet boundary condition at the top of the domain
  dirichletBottom: true, // Set a Dirichlet boundary condition at the bottom of the domain
  dirichletLeft: true, // Set a Dirichlet boundary condition at the left side of the domain
  dirichletRight: true, // Set a Dirichlet boundary condition at the right side of the domain
  dirichletValueTop: 0.0, // Dirichlet boundary condition at the top of the domain
  dirichletValueBottom: 1.0, // Dirichlet boundary condition at the bottom of the domain
  dirichletValueLeft: 1.0, // Dirichlet boundary condition at the left side of the domain
  dirichletValueRight: 1.0, // Dirichlet boundary condition at the right side of the domain
};

Then, we call the CFDScript library to solve the 2D Laplace equation. 

// Script2: 2D Laplace solution
import { CFDScript } from './scripts/CFDScript.js';
window.addEventListener('DOMContentLoaded', (event) => {
  // Call the CFDScript library and retrieve the solution
  let { u, nx, ny, axpt, aypt } = CFDScript(nex, ney, xlast, ylast, boundaryConditions);
});

The visualization of the results can be performed by using a Graphing Library such as Plotly.js. See the example page for a full application.
